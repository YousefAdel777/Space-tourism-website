# Space Tourism Website
A simple space tourism website made using React, Vite, Framer motion and TailwindCSS.

## Features
- 4 different pages.
- Smooth animations using framer-motion.

## Made using
- React.JS
- TailwindCSS
- Vitest
- react-router
- framer-motion
